139 /* The following parameters hold the values of 4096 divided by the corresponding
140 individual quantization values. These values are needed to get around
141 actually dividing the Y, Cb, and Cr values by the quantization values.
142 Instead, you can multiply by the values below, and then divide by 4096
143 to get the same result. And 4096 = 2^12, so instead of dividing, you can
144 just shift the bottom 12 bits out. This is a lossy process, as 4096/Q divided
145 by 4096 doesn't always equal the same value as if you were just dividing by Q. But
146 the quantization process is also lossy, so the additional loss of precision is
147 negligible. To decrease the loss, you could use a larger number than 4096 to
148 get around the actual use of division. Like take 8192/Q and then divide by 8192.**/
